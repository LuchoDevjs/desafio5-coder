#### Configuracion

  Ya ejecutado, crear un archivo llamado ".env" y dentro escribir lo siguiente

```
PORT=8080
NODE_ENV=local
NODE_URL="http://localhost:8080/"
```

Por ultimo ejecutar el comando

```
npm start
```
